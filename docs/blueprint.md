# **App Name**: Funda Finder

## Core Features:

- Interactive Questionnaire: Guides the user through a series of questions to gather their property preferences (location, price range, size, number of bedrooms, energy label and construction period).
- Query Formulation Tool: AI tool transforms user answers into a structured query for Funda's search parameters. AI will validate the values provided for numerical and list inputs.
- Web Scraper Integration: Connects to Funda's search to execute queries, mimicking the website's URL structure for filtering.
- Results Display: Presents scraped search results in an organized, readable manner, showcasing properties that match the user's specified criteria.
- Parameter persistence: Ability to save user selected parameters into local storage or database

## Style Guidelines:

- Primary color: Soft blue (#7EC4CF) to create a calm and trustworthy impression.
- Background color: Very light blue (#E0F4F7), providing a clean, unobtrusive backdrop for the property listings.
- Accent color: Warm orange (#FFB347) to highlight interactive elements such as call-to-action buttons and important filters.
- Font pairing: 'Inter' (sans-serif) for body text providing a modern, clean reading experience, and 'Space Grotesk' (sans-serif) for headlines offering a techy touch.
- Use clear and concise icons to represent property features like number of bedrooms, floor area, and energy label, improving readability and user experience.
- Design a grid-based layout for property results to maintain a consistent and visually appealing presentation.
- Incorporate subtle transitions and animations when loading results and navigating through the questionnaire to create a smoother user experience.
